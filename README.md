# Password-Strength-Checker
A Python automation tool that analyzes the strength of passwords and provides instant feedback to help users create stronger, more secure passwords.

In today's digital world, weak passwords are one of the biggest security risks. This tool checks your password against common security rules and gives it a strength rating — so you always know how safe your password really is.
It checks for:

✅ Minimum length (e.g. 8+ characters)
✅ Uppercase and lowercase letters
✅ Numbers
✅ Special characters (e.g. @, #, !, $)
✅ Common/weak passwords (e.g. password123, 123456)

Strength levels it returns:

🔴 Weak — easily guessable
🟡 Moderate — needs improvement
🟢 Strong — good to go!

Python 3.x
